# Book Tracker

Автор: Корыстылёва Анна
Дата: 04.05.2026

## Описание
Приложение для учёта прочитанных книг.  
Позволяет добавлять книги (название, автор, жанр, количество страниц),  
фильтровать по жанру и по минимальному количеству страниц,  
сохраняет данные в JSON-файл.

## Инструкция по запуску
1. Убедитесь, что установлен Python 3.
2. Скачайте файл book_tracker.py.
3. Запустите:
   ```bash
   python book_tracker.py
